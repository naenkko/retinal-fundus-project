# Классификация заболеваний глазного дна по снимкам (fundus images)

Преддипломный проект по анализу медицинских изображений глазного дна с помощью методов глубокого обучения. Цель работы — построить модель классификации заболеваний и проверить гипотезу о том, **улучшает ли предварительная обработка снимков качество распознавания**.

## О чём проект

В работе используются три набора снимков глазного дна:

| Датасет | Задача | Классы |
|---|---|---|
| **eye_diseases** | классификация типа заболевания | 4 класса: норма, диабетическая ретинопатия, катаракта, глаукома |
| **glaucoma** | бинарная классификация | 2 класса: наличие / отсутствие глаукомы (RG / NRG) |
| **IDRiD** | степень диабетической ретинопатии | 5 классов (DR-grade 0–4) |

Всего после очистки — **14 210 изображений**.

## Что сделано

### 1. Разведочный анализ данных (EDA)
- анализ распределения классов, размеров и качества изображений;
- выявление дисбаланса классов и особенностей каждого датасета.

### 2. Предобработка изображений
Реализован конвейер предобработки снимка:
1. обнаружение и обрезка области поля зрения (FOV-crop);
2. приведение к единому размеру 512×512;
3. повышение локального контраста методом **CLAHE**;
4. нормализация значений пикселей.

Предобработка применена ко всем изображениям; обработанные версии сохранены **отдельно**, оригиналы сохранены без изменений.

### 3. Модель классификации
- архитектура **ResNet-18** (transfer learning, веса ImageNet);
- **отдельная модель на каждый датасет** (под разное число классов);
- учёт дисбаланса классов через взвешенную функцию потерь;
- обучение с ранней остановкой и сохранением лучшей по валидации модели.

### 4. Сравнение «до / после обработки»
Для каждого датасета обучены две модели с идентичной архитектурой и гиперпараметрами — на **необработанных (RAW)** и **обработанных (PROC)** снимках. Оценка на отложенной тестовой выборке по метрикам: accuracy, macro-F1, balanced accuracy, ROC-AUC, матрицы ошибок.

## Основные результаты

| Датасет | Режим | Accuracy | Macro-F1 | ROC-AUC |
|---|---|---|---|---|
| eye_diseases | RAW | 0.916 | 0.915 | 0.986 |
| eye_diseases | PROC | 0.905 | 0.904 | 0.986 |
| glaucoma | RAW | 0.923 | 0.923 | 0.973 |
| glaucoma | PROC | 0.914 | 0.914 | 0.966 |

**Вывод:** гипотеза об улучшении качества за счёт предобработки **не подтвердилась** — метрики на обработанных данных оказались не выше, чем на необработанных (различия ≈ 1 п.п., в пределах статистической погрешности). Вероятная причина — ResNet-18, предобученная на ImageNet, уже устойчива к различиям в контрасте и освещении.

> Датасет IDRiD исключён из итогового сравнения из-за сильного переобучения (малый объём данных и сдвиг распределения классов между train и test).

## Стек технологий

- **Python 3.9**
- **PyTorch**, **TorchVision** — обучение модели
- **OpenCV** — обработка изображений
- **scikit-learn** — метрики качества
- **NumPy**, **Pandas**, **Matplotlib** — анализ и визуализация
- **Jupyter Notebook**

## Воспроизводимость

- зафиксировано случайное зерно (seed = 42);
- единые train/val/test-сплиты для всех экспериментов;
- проверка на утечку данных между сплитами.
